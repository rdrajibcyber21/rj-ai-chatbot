# 🤖 AI Chatbot (FastAPI + Ollama)

A full-stack AI chatbot with chat history, voice input, and modern UI.

---

## 🚀 Features

- Chat with AI (Llama3 via Ollama)
- Save chat history (SQLite)
- Rename & delete chats
- Voice input (speech-to-text)
- Sidebar chat system

---

## 🛠️ Tech Stack

- FastAPI (Backend)
- SQLite (Database)
- HTML, CSS, JavaScript (Frontend)
- Ollama (AI Model)

---

## 📂 Project Structure

backend/ → API & database  
frontend/ → UI  

---

## ⚙️ Run Project

### Backend
cd backend  
source venv/bin/activate  
uvicorn main:app --reload  

### Frontend
cd frontend  
python3 -m http.server 5500  

Open: http://localhost:5500
